A web-based astronomy tool that calculates the altitude and azimuth
of celestial objects (planets, Moon) from a user-defined location and time.

---

## Features

- Location lookup via geopy
- Precise ephemerides via Skyfield (JPL DE421)
- Web interface powered by Flask

---

## Setup

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py
```
